A complete config file should have at least the following three components in this order:
1. One line with an integer n: Number of machines
2. One line for each machine: Alphas for each machines 
3. One line with an integer: The index of mechanism used for the scheduler.

Some optional parameters can also be filled in:
4. For each machine, write a randomized decider function with format "lambda y:<f(y)>"
